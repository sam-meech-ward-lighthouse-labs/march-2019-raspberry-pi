## Commands

ssh onto the raspberry pi

```shell
ssh pi@192.168.88.28
```

Copy the current directory to the raspberry pi in the `~/oven` directory.

```shell
rsync -avz --exclude node_modules . pi@192.168.88.28:~/oven
```
